Self-Healing Biopolymer Film Prediction System

This project is a web-based application developed using Streamlit that predicts the healing efficiency and properties of self-healing biopolymer films made from natural materials.

🔬 Project Overview

Self-healing materials are an emerging area in biomedical and sustainable engineering. This project focuses on estimating the healing efficiency of biopolymer films based on material composition and environmental conditions.

⚙️ Features

- Input material composition:
  - Chitosan
  - Gelatin
  - Starch
  - Aloe Vera
  - Glycerol
- Environmental parameters:
  - Temperature
  - Humidity
  - pH level
- Damage type selection:
  - Scratch / Cut / Crack
- Output predictions:
  - Healing Efficiency (%)
  - Material Strength
  - Biodegradability

🧠 Logic Used

A rule-based system is implemented to estimate how different parameters influence the healing efficiency of biopolymer films.

💻 Technologies Used

- Python
- Streamlit
- Pandas
- Matplotlib

🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
